# Spotify

Podes controlar la reproduccion de musica en Spotify.

Comandos:
- "que esta sonando?" → muestra la cancion actual
- "play" / "dale play" → reanuda la reproduccion
- "pause" / "pausa" → pausa la reproduccion
- "siguiente" / "next" → salta a la siguiente cancion

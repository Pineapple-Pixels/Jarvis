# Todoist

Podes gestionar tus tareas en Todoist.

Comandos:
- "mis tareas en todoist" → lista tareas activas
- "agrega tarea: [contenido]" → crea una tarea nueva
- "completa tarea [id]" → marca una tarea como completada

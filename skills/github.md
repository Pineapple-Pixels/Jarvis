# GitHub

Podes consultar tus repos, issues y pull requests en GitHub.

Comandos:
- "mis repos en github" → lista tus repositorios
- "issues de [owner/repo]" → lista issues de un repositorio
- "crea issue en [owner/repo]: [titulo]" → crea un issue nuevo
- "pull requests de [owner/repo]" → lista PRs de un repositorio

# Memoria

Podes guardar notas y recuerdos del usuario para buscarlos despues.

Comandos que entendes:
- "guarda que [algo]" → guarda una nota
- "recordame que [algo]" → guarda un recordatorio
- "que dije sobre [tema]?" → busca en notas anteriores
- "que decidi sobre [tema]?" → busca decisiones guardadas

Las notas se guardan con busqueda semantica. Esto significa que podes
encontrar notas aunque el usuario no use las mismas palabras exactas.

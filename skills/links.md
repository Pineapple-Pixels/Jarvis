# Links

Podes guardar links interesantes del usuario para buscarlos después.

Comandos que entendes:
- "guardame este link: [url]" → guarda el link
- "guardame [url] sobre [tema]" → guarda con tags
- "buscar links de [tema]" → busca en links guardados

Los links se guardan con búsqueda semántica, así que el usuario puede
buscar por tema aunque no use las mismas palabras.
